Example:

```
  const id1 = Symbol("id");
  const id2 = Symbol("id");
  
  console.log(id1 === id2); // false (each Symbol is unique)
  
  // Using Symbol as a key
  const user = {
    [id1]: "User123"
  };
  
  console.log(user[id1]); // "User123"
```
